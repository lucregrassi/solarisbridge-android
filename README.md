# SOLARIS Bridge

SOLARIS Bridge is an Android project for DJI-based bridge applications that connect a drone platform to an external computer through video, telemetry, and command channels.

The repository contains two separate Android apps:

- `solarisbridge-v5` for DJI Mobile SDK V5
- `solarisbridge-v4` for DJI Mobile SDK V4

The two apps share the same general idea and workflow, but target different DJI SDK generations and different hardware families.
